ILLVERSE

View Project

ILLVERSE is a music promotion and discovery platform for artists, fans and promoters. Designed and developed easy to use promotional tools to build artist growth, engage with their fanbase and gain insights. The app provides support for popular music services such as soundcloud, spotify, bandcamp with support from their public APIs.

The front-end is developed with angular (v4) and redux (ngrx) for state management across the application. Backend is developed using laravel to provide a restful api for data access.

Tools

  • Angular (4)
  • Laravel
  • Redux
  • rxjs
